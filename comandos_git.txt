git init . 			 -> Para transformar uma pasta em repositorio git
git status 			 -> Para checar o status dos documentos
git add "nome do arquivo" ou "." -> Adiciona a fila para ser comitado em seguida
git commit -m "mensagem" 	 -> Comita o arquivo e diz quais foram as mudanças feitas
git log                          -> Mostra o historico dos commits feitos

## FLUXO DE TRABALHO

1. criar o diretorio do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo> ou <.> (para pegar a pasta inteira)
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3.
